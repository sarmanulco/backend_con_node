Métodos HTTP

Hay 4 métodos HTTP

  • POST: Se utiliza para crear un recurso
  • GET: Se utiliza para consultar uno o varios recursos
  • PUT: Se utiliza para modificar uno o varios recursos
  • DELETE: Se utiliza para eliminar uno o varios recursos

Postman nos sirve para probar nuestras apis

Método GET

  • Función INDEX: Obtener todos los elementos de un recurso
  • Función SHOW: Obtener un elemento de un recurso
  • Función EDIT: Obtener el formulario de creación de un recurso

Otros métodos

  • Función create (POST): Es la función en la que implementamos la creación de un recurso
  • Función update (PUT): Es la funcion en la que implementamos la actualización de un recurso
  • Función destroy (DELETE): Es la función en la que implementamos la destrucción de un recurso
